# JavaScript & TypeScript Learning Journey 🚀

A curated collection of programs I'm writing to learn JavaScript and TypeScript, from the ground up.

## Structure

```
learn-playwright/
├── README.md                  ← you are here
├── chapter-01-basics/         ← JavaScript/TypeScript fundamentals
│   └── 01-hello-world.js
└── iq-notes/                  ← concept reference cards
    ├── README.md               ← template for adding new notes
    └── source-vs-bytecode-vs-binary.md
```

- **`chapter-*`** — Hands-on programs grouped by topic. Each chapter builds on the last.
- **`iq-notes/`** — Quick-reference notes comparing related concepts side-by-side.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18+)

## Usage

```bash
node chapter-01-basics/01-hello-world.js
```
