# IQ Notes — Concept Comparison Template

Use this template to create structured comparison notes for any technical concept. Each note follows the same visual style as `source_byte_binary_code.md`.

## Template Prompt

```
Explain [YOUR CONCEPT] in a table view with clear comparison columns.
Use a simple code/file example to walk through each layer.
Save it as a .md file inside the IQ_Notes folder.
Follow the same visual style as source-vs-bytecode-vs-binary.md — breakdown table,
example walkthrough, pipeline diagram, and TL;DR.
```

## Markdown Structure to Follow

Every note should include:

1. **Heading** — `# [Concept Name]`
2. **Code Example** — a simple snippet
3. **Comparison Table** — columns: Aspect, Item A, Item B, Item C...
4. **Real-world Walkthrough** — step-by-step flow using the example
5. **Pipeline Diagram** (text-based) — visual flow from start to end
6. **TL;DR** — one-line summary
